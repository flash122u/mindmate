# 开发指南

## 环境准备
| 工具 | 版本 |
| --- | --- |
| Python | >=3.11 |
| pip | 最新即可 |
| Git | 可选（当前无 git 仓库） |

## 常用命令
| 操作 | 命令 |
| --- | --- |
| 安装依赖 | `pip install -e .` |
| 安装开发依赖 | `pip install -e ".[dev]"` |
| 安装 MCP 扩展 | `pip install -e ".[mcp]"` |
| 本地启动 | `python -m mindmate.main` |
| 运行全部测试 | `pytest tests/ -q` |
| 运行单文件测试 | `pytest tests/test_trace.py -q` |
| 代码检查 | `ruff check mindmate/ tests/` |
| 自动修复 | `ruff check --fix mindmate/ tests/` |

## 配置文件清单
| 文件 | 作用 |
| --- | --- |
| `.env` | 环境变量：API Key、模型、端口、主动推送开关、城市 |
| `pyproject.toml` | 项目元数据、依赖声明、ruff 配置、pytest 配置 |
| `memory/SOUL.md` | Agent 人格核心（人物小传、说话习惯、雷区） |
| `memory/MEMORY.md` | 长期事实记忆（LLM 自动维护） |
| `memory/memory.db` | SQLite 数据库（自动创建，8 张表） |
| `CLAUDE.md` | Claude Code 项目记忆文件（启动协议、开发流程） |

## 环境变量 (.env)
| 变量 | 默认值 | 说明 |
| --- | --- | --- |
| `DEEPSEEK_API_KEY` | (必填) | DeepSeek API 密钥 |
| `DEEPSEEK_BASE_URL` | `https://api.deepseek.com/v1` | API 地址 |
| `DEEPSEEK_MODEL` | `deepseek-chat` | 模型名 |
| `OPENAI_API_KEY` | - | 备用 OpenAI Key |
| `HOST` | `0.0.0.0` | 监听地址 |
| `PORT` | `19876` | 监听端口 |
| `HOME_CITY` | `杭州` | Agent 所在城市（天气查询用） |
| `TOOLS_ENABLED` | `true` | 是否启用工具调用 |
| `INNER_LIFE_ENABLED` | `true` | 是否启用每日日记/梦境生成 |
| `INNER_LIFE_HOUR` | `3` | 每日内在生活触发时间（24h） |
| `MCP_SERVERS` | - | JSON 格式的 MCP server 配置 |

| `http://localhost:19876/` | 聊天页面 |
