# 业务知识

## 项目背景
<!-- TODO: 一句话描述项目 + 解决什么问题 + 服务谁 -->
MindMate 是一个"伙伴型"而非"工具型"的 AI 心理陪伴 Agent。目标是模拟真人好友的对话质感——不秒回、不说教、有情绪、记得事——为需要情感倾诉的用户提供一个随时在线、有记忆、懂分寸的聊天伙伴。

## 领域术语
| 术语 | 代码名 | 业务含义 |
| --- | --- | --- |
| 情绪锚点 | `emotion_anchor` | 从对话中提取的"事件-情绪-触发线索-效价"四元组，用于后续相似场景的共情回忆 |
| 效价 | `valence` | 情绪的正负面程度，-1.0（极负面）到 1.0（极正面），驱动关系阶段变化 |
| 关系阶段 | `relationship stage` | 三段渐进式：初识 → 朋友 → 信赖，影响回复的亲密度和主动推送策略 |
| 雷区防御 | `DefenseMechanism` | 当用户施压/质疑 Agent 身份时，Agent 的温和回避机制，赋予 Agent 心理边界 |
| 内在生活 | `inner_life` | Agent 的私密日记和梦境生成（子 Agent），默认不进对话，仅在深度交心场景选择性分享 |
| 能量模型 | `EnergyModel` | 控制主动推送时机的四维规则：沉默阈值、冷却、安静时段、每日上限 |
| 危机检测 | `crisis_detect` | 分级规则匹配（medium/high），命中 high 时触发预警并生成安抚话术 |
| 消息拆分 | `split_message` | 将 LLM 回复按句号/换行拆成多条短消息，模拟微信聊天节奏 |
| 记忆压缩 | `memory_consolidation` | 当历史消息超出窗口（20 轮），将早期对话通过 LLM 压缩合并入 MEMORY.md |
| 记忆遗忘 | `forget` | 久远且低关联的情绪锚点随时间逐步弱化效价，低于阈值后删除 |
| 意图路由 | `IntentRouter` | LLM 调用前的前置分类器，risk/functional/identity 三类走预写模板跳过 LLM |

## 核心业务规则

### 关系演进规则
<!-- TODO: 确认具体阈值 -->
- 关系阶段变化由近期情绪锚点的平均效价驱动
- agent/loop.py 中 `_update_relationship()` 根据锚点均值判断推进/回退

### 主动推送规则
- 每 60 秒检查一轮所有活跃用户
- 四维能量模型全部通过才触发主动问候
- 主动消息由 LLM 结合用户近期上下文生成，非硬编码模板

### 危机预警规则
- tools/crisis_detect.py 中的 `_RISK_PATTERNS` 定义匹配规则
- high 级别：自杀/自残等极高风险信号 → 立即预警 + 心理援助热线
- medium 级别：崩溃/绝望等中度风险 → 注入关切指引

### 记忆生命周期
1. 每条对话 → `history` 表（不可篡改时间线）
2. LLM 提取情绪锚点 → `emotion_anchors` 表
3. 超窗口对话 → MemoryConsolidator 压缩 → `MEMORY.md`
4. 久远低关联锚点 → ForgetAgent 衰减 → 低于阈值删除

### 风格约束（硬编码在 AgentLoop._STYLE_RULES）
- 回复长度：1-3 句
- 禁止：客服腔、分点列条、说教、描述动作神态
- 允许：语气词、反问、跑题、表达自己情绪
