# Step 4：主动行为系统 ✅

## 目标

让小暖在你沉默时会主动找你——但像真朋友一样**有分寸**，不像骚扰。

## 已实现

### 能量模型（agent/energy.py）
决定"什么时候该主动开口"：
- `should_reach_out()` 综合判断：
  - 沉默时长 ≥ 阈值（默认 30 分钟）
  - 不在冷却中（默认两次间隔 ≥ 2 小时）
  - 不在安静时段（默认 23:00–08:00）
  - 未超每日上限（默认 4 次）
- `on_user_message()` 重置沉默计时
- `mark_proactive()` 记录开口、进入冷却
- 与 AgentLoop 共享同一实例

### 主动消息生成（agent/loop.py: generate_proactive）
- 用 LLM + 人格 + 历史上下文生成自然开口（不是硬编码模板）
- `_proactive_nudge()` 按时段/随机意图引导：问候/分享碎屑/关心/回顾旧事
- 走同一分段投递链路，存入历史保持连贯

### 主动循环（proactive/loop.py）
- 定时检查能量 → 满足则调 generate_proactive → 标记冷却

## 实测效果

```
（小暖记得上次聊的考研）
🌿 对了，你上次说的考研方向，后来决定了没呀？
```
不是模板"今天过得怎么样"，而是有上下文、像真朋友的关心。

## 关键设计

避免"像骚扰"的核心是**分寸机制**：刚聊完不发、半夜不发、有冷却、有每日上限。
主动消息走 LLM 生成而非模板，才能有上下文和人格。
